# Copy Revamp Plan v2 — The Brand OS Intensive
**Lens:** Bold. Direct. Conversion-first. Every line moves someone toward registering.
**Audience:** Founders, operators, and executives who want to be publicly known — thought leaders, sought-after speakers, the name their category follows. They're already expert. Their public presence doesn't show it yet.

**Rule for this pass:** No hedging. No "could." No "might." Only declarative, present-tense, conversion-driving copy. If a sentence doesn't make someone feel something or click something, cut it.

---

## POSITIONING IN ONE LINE
> **You're already the expert. This afternoon makes you the name.**

Everything else on the site proves this claim and makes signing up feel inevitable.

## CONVERSION COPY RULES (applied throughout)

| Rule | Example |
|---|---|
| Imperative verbs | Claim. Stop. Walk out. Build it. |
| Present tense, active voice | "He installs" not "is installed" |
| Binary framing | You have the system or you don't |
| No hedging | Never "might" "could" "possibly" |
| Specificity as confidence | "50+" "15 assets" "30 min/week" "one afternoon" |
| Close every section with a pull toward action | "The seat is there. Take it." |
| Use "you" 2x more than "we" | Keeps focus on the attendee's outcome |
